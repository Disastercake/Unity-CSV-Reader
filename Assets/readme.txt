How to test this demo:
1. Put a CSV file (comma delimeter) in the Resources.
2. Open Assets >> CSV >> Test CSV Reader
3. Put the file name in the textbox and press the button.
4. Read the output in the Console.

How to use the CSV reader in your project:
1. Import in the entire Assets/LumenWorks directory.
2. Import the file: Scripts/CsvData.cs
3. Import the file: Scripts/CsvHandler.cs
